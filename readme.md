# clz32

> ES2015 [`Math.clz32()`](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Math/clz32) [ponyfill](https://ponyfill.com)


## Install

```
$ npm install --save clz32
```


## Usage

```js
var clz32 = require('clz32');

clz32(1000);
//=> 22
```
